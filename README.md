#  IPL Sports Data Analytics Dashboard

An interactive **IPL Sports Data Analytics Dashboard** built using **Python, Streamlit, Pandas, Matplotlib, and Plotly**. This project provides comprehensive insights into Indian Premier League (IPL) matches, teams, players, and season statistics through an easy-to-use web interface.

---

##  Project Overview

The IPL Sports Data Analytics Dashboard enables users to analyze historical IPL data using interactive charts and visualizations. It helps users explore team performance, player statistics, match outcomes, and tournament trends.

---

##  Features

- Interactive Streamlit Dashboard
- Team Performance Analysis
- Data Cleaning & Preprocessing
-  Interactive Visualizations
-  Dynamic Filters
-  Win Percentage Analysis

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Plotly
- Seaborn
- VS Code

## 📥 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/IPL-Sports-Data-Analytics.git
```

### Navigate to Project

```bash
cd IPL-Sports-Data-Analytics
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
